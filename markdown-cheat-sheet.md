# Markdown Cheat Sheet Personalizado por mateus-o-m

Mais informações disponíveis em [The Markdown Guide](https://www.markdownguide.org)!

## Sintaxe básica

Todas as aplicações Markdown suportam os elementos abaixo,
que são o design original de John Gruber

### Cabeçalho

# H1
## H2
### H3

### Negrito

**texto em negrito**

### Italico

*texto em itálico*

### Citação

> Citação

### Lista ordenada

1. Primeiro item
2. Segundo item
3. Terceiro item

### Lista não ordenada

- Primeiro item
- Segundo item
- Terceiro item

### Código

`código`

### Régua horizontal

---

### Link

[Markdown Guide](https://www.markdownguide.org)

### Imagem

![alt text](https://www.markdownguide.org/assets/images/tux.png)
